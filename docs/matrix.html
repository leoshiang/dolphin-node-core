逆矩陣 A−1

逆矩陣（inverse matrix），又稱乘法反方陣、反矩陣。在線性代數中，給定一個 n 階方陣，若存在一 n 階方陣 B，使得 AB=BA=In ，其中 In 為 n 階單位矩陣，則稱 A 是可逆的，且 B 是 A 的逆矩陣，記作 A−1。

以下列的矩陣為例，最右邊為逆矩陣的結果：
