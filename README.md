# dolphin-node-core

`dolphin-node-core` 是一個 node.js 的套件，由向皓田開發，採用 MIT 授權。此套件包含了一些常用的 JavaScript 物件集合，例如：型別處理、向量、矩陣、位元陣列。

## 特色

* Type 是將一些型別判斷的功能的集合。
* Bits 是一個最佳化過的位元陣列，當你需要儲存大量的 Boolean 值時，Bits 所佔用的空間是一般陣列的 1/24。
* Vector 是多維的向量。
* Matrix 是二維矩陣，提供基本的矩陣運算。
